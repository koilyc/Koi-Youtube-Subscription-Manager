# Koi YouTube Subscription Manager

A Chrome Extension that helps you organize your YouTube subscriptions into custom folders — clean, fast, and styled to match YouTube's native UI.

## Features

- **Folder Management** — Create, rename, and delete folders with custom colors to categorize your subscriptions
- **Drag & Drop** — Move channels between folders by dragging
- **List / Grid View** — Switch between list and grid display modes; your preference is remembered across sessions
- **Search** — Instantly filter channels by name
- **One-Click Sync** — Pull your latest YouTube subscriptions via the YouTube Data API
- **YouTube-Native Design** — Dark theme UI that blends seamlessly with YouTube's look and feel
